# 🐙🥦 stunning-octo-broccoli

> **学习编程 · 传递善意 · 共建开源社区** — Just Learn It

一个面向编程初学者的 **8 模块交互式学习平台**。精美设计、零基础友好、所有收益 100% 捐赠西兰花公益计划。

## 🧭 八大学习模块

像章鱼的八条触手，每一条带你掌握编程的一个维度：

| # | 模块 | 内容 | 难度 |
|---|------|------|------|
| 🌱 | **Seed · Hello World** | 写出第一行代码，理解程序是什么 | 入门 |
| 📦 | **Sprout · 变量与类型** | 用变量存储数据，探索数据类型 | 基础 |
| 🔄 | **Branch · 条件与循环** | 让程序做判断、重复执行 | 核心 |
| 🌿 | **Leaf · 函数** | 封装可复用代码，像搭积木一样编程 | 核心 |
| 📋 | **Tree · 数据结构** | 用数组和对象构建信息之树 | 进阶 |
| 🐛 | **Debug · 调试技巧** | 像侦探一样排查 Bug | 技巧 |
| 🌐 | **Web · 网页基础** | HTML + CSS + JS，构建第一个网页 | 项目 |
| 🤝 | **Harvest · 开源贡献** | 加入开源社区，提交第一个 PR | 实战 |

## 🚀 快速开始

### 在线体验

直接访问项目 demo 页面（部署后替换为实际 URL）

### 本地运行

```bash
# 克隆仓库
git clone https://github.com/beforewind6/stunning-octo-broccoli.git

# 进入项目目录
cd stunning-octo-broccoli

# 用任意 HTTP 服务器启动，例如：
python -m http.server 8080
# 或
npx serve .
```

然后打开浏览器访问 `http://localhost:8080`

## 🛠 技术栈

- **HTML5** — 语义化结构
- **CSS3** — 暗色主题 · CSS 变量 · Grid/Flexbox · 响应式设计
- **Vanilla JavaScript** — 零依赖 · 交互模块 · 代码游乐场 · 进度追踪

> 故意不引入框架 —— 降低初学者门槛，用最朴素的技术栈展现编程之美。

## 🎮 Demo 功能

| 功能 | 说明 |
|------|------|
| **模块学习卡片** | 8 个交互卡片，点击标记完成状态 |
